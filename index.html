SoulMask サーバー設定エディタ(GameXishu.json)

ブラウザ完結型編集ツール  |  最終更新: 2026-05-16  |  対応版: SoulMask 1.0 Update Patch #10 (BuildID 23229545)

ファイル未読み込み — JSONファイルを読み込むか、直接値を編集してください
📖 このツールの目的と使い方

🎯 目的

SoulMask 専用サーバー (Dedicated Server) の動作を制御する設定ファイル GameXishu.json を、JSON を直接書き換えることなく Web 画面で安全に編集するためのツールです。経験値倍率、ドロップ率、PvP/PvE 設定、侵略、配下数など 200 以上のパラメータを 日本語ラベル・効果説明・推奨値 付きで編集できます。

🚀 使い方の流れ

  1. 既存設定を読み込む: 右上の「📂 JSONファイルを読み込む」から、サーバーの現在の GameXishu.json を選択
    パス例: <サーバーフォルダ>\WS\Saved\GameplaySettings\GameXishu.json
  2. 値を編集: タブ・カテゴリを開いて各パラメータを変更。各項目には影響説明・範囲・推奨値が表示されます
  3. 0/2 タブへ反映: 推奨は プリセット 1 で編集 → 「→ 0/2 両方へ反映 (推奨)」ボタンで全タブを同一値に揃える
  4. JSONをダウンロード: 右上の「💾 JSONをダウンロード」で GameXishu.json をダウンロード
  5. サーバーへ反映: サーバーを停止 → ダウンロードしたファイルを元の場所に上書き → サーバー再起動

🗂 プリセット 0 / 1 / 2 の使い分け

  • プリセット 0: ローカル / シングルプレイ用と推測される係数
  • プリセット 1 ★メイン: 専用サーバーで実際に効く係数 — まずここを編集
  • プリセット 2: 用途未公開 (PvP / 特殊モード予備枠)

3 セクションの正確な使い分けは公式未公開のため、コミュニティ標準では 0 / 1 / 2 を同じ値に揃える運用が推奨されています。

⚠️ 注意: サーバー起動中に GameXishu.json を上書きすると、サーバー終了時に古い値で上書きされて編集内容が消えることがあります。必ずサーバーを停止してからファイルを置き換えてください。
